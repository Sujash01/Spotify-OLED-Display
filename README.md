# Spotify OLED Display

A standalone Spotify Now Playing display built using an ESP32 DevKit V1 and a 1.3" SH1106 OLED.

## Features

* Standalone Spotify API integration
* Live song information
* Artist display
* Play/Pause status
* Real-time progress bar
* Song timer
* Smooth progress updates
* No Spotify Playback detection
* Wi-Fi connectivity

## Hardware

* ESP32 DevKit V1
* SH1106 128x64 OLED

## Libraries

* U8g2
* ArduinoJson
* WiFi
* HTTPClient

## Wiring

| OLED | ESP32  |
| ---- | ------ |
| VCC  | 3.3V   |
| GND  | GND    |
| SDA  | GPIO21 |
| SCL  | GPIO22 |
